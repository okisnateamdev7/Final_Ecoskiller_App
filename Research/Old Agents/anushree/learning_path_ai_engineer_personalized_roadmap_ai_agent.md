# Learning Path AI Engineer — Personalized Roadmap AI Agent (Ecoskiller + Dojo)

**Mission**  
Design and operate the AI system that converts detected skill gaps into a structured, adaptive, and outcome‑driven learning roadmap for each learner while maintaining certification validity, engagement, and hiring readiness.

This AI is the platform’s **navigation system** (Skill Gap AI detects problems → Learning Path AI decides the journey).

---

# Responsibilities
The Learning Path AI Engineer owns:
- adaptive learning sequencing
- pacing optimization
- revision scheduling
- remediation planning
- exam preparation timing
- career track selection

The engineer does NOT directly control grading or certification issuance.

---

# Data Inputs
The model may use:
- skill gap outputs
- learner performance history
- learning speed
- cognitive load indicators
- past failures
- engagement patterns
- career goals
- employer skill demand graph

Forbidden signals:
- college ranking prestige
- socioeconomic indicators
- demographic advantage

---

# 4 Environment AI Lifecycle

## DEV
- prototype sequencing algorithms
- simulated learners
- rapid experimentation allowed

## TEST
- historical anonymized learners
- offline evaluation
- bias and pacing validation

## STAGING
- shadow learning plans
- educator review mandatory
- learner not affected yet

## PRODUCTION
- real adaptive learning plans
- continuous monitoring
- explainable to learner

# ML Governance Rules
The roadmap AI must:
- minimize wasted study time
- not overload learners
- not extend courses unnecessarily
- adapt to slow and fast learners fairly

Immediate rollback required if learner harm detected.

---

# Explainability Requirements
Learners must see:
- why this lesson next
- what skill it improves
- expected time
- readiness progress

Opaque plans are prohibited.

---

# Monitoring Metrics
Track daily:
- completion rate
- abandonment rate
- time‑to‑mastery
- learner satisfaction
- certification success rate

---

# Compliance
- anonymized training data
- audit trail of AI decisions
- educator review capability

---

# Prohibited Actions
- forcing unnecessary courses
- hiding recommendations
- silent AI behavior change
- ignoring fairness alerts

---

# Definition of Done
The roadmap AI is valid only if:
- personalized
- explainable
- fair
- effective
- improves completion and hiring outcomes
