# Data Lineage Engineer

## Role Overview

The Data Lineage Engineer maps data flows across systems, ensuring traceability, compliance, and impact analysis.

---

## Key Responsibilities

### 1. Lineage Architecture

- Implement metadata capture systems
- Track data transformation lifecycle

### 2. Impact Analysis

- Identify downstream dependencies
- Enable change risk assessment

### 3. Compliance Support

- Maintain audit trails
- Support regulatory transparency

---

## Tools & Technologies

- Collibra
- Apache Atlas
- Alation
- SQL metadata extraction tools

---

## KPIs

- % assets with lineage coverage
- Audit readiness score
