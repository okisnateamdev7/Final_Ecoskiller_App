# Supply Chain Security Engineer

## Role Overview

Ensures security of software supply chain, third-party dependencies, and
CI/CD pipelines.

## Responsibilities

-   Dependency scanning
-   SBOM management
-   CI/CD security validation
-   Third-party risk analysis
-   Build integrity verification

## ML Layer

-   Dependency risk scoring
-   Vulnerability trend forecasting
-   Malicious package detection
-   Build anomaly detection

## Governance

Mandatory scanning before production deployments.
