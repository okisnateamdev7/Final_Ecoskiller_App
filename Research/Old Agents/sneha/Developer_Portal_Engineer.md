# Developer Portal Engineer --- Public Developer Platform

## Role Overview

Builds and maintains public developer portals, onboarding flows, API key
management, and documentation hosting.

## Responsibilities

-   Portal UI/UX management
-   API key issuance workflows
-   Rate limit dashboards
-   Developer analytics integration
-   OAuth integration

## ML Layer

-   Developer engagement scoring
-   Behavioral clustering
-   Portal performance prediction
-   Abuse detection (Isolation Forest)

## Governance

Sandbox APIs in DEV, controlled publishing to PRODUCTION.

## Validation

Git-ready Markdown structure.
