# Data Quality Engineer

## Role Overview

The Data Quality Engineer ensures reliability, accuracy, completeness, and consistency of enterprise data across systems.

---

## Key Responsibilities

### 1. Data Quality Framework

- Define DQ standards & policies
- Implement validation rules
- Establish anomaly detection systems

### 2. Monitoring & Automation

- Build automated data validation pipelines
- Monitor data drift & inconsistencies
- Implement reconciliation processes

### 3. Root Cause Analysis

- Investigate data discrepancies
- Work with engineering teams to fix upstream issues

### 4. Reporting

- Maintain DQ dashboards
- Track quality KPIs

---

## Tools & Technologies

- Great Expectations
- dbt tests
- SQL, Python
- Airflow
- Data observability tools

---

## KPIs

- Data accuracy %
- Incident resolution time
- Data completeness score
