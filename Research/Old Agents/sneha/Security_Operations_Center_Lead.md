# Security Operations Center (SOC) Lead

## Role Overview

Leads enterprise SOC operations, incident response governance,
monitoring strategy, and threat intelligence coordination.

## Core Responsibilities

-   24/7 monitoring governance
-   SIEM strategy & tuning
-   Incident response leadership
-   Threat intelligence integration
-   Escalation & SLA enforcement
-   Security metrics reporting

## ML Intelligence Layer

-   SIEM log clustering (K-Means)
-   Threat anomaly detection (Autoencoders)
-   Incident severity prediction (Gradient Boosting)
-   Alert noise reduction (Isolation Forest)
-   Attack pattern recognition (Deep Learning)
