# Cloud FinOps Analyst --- Unit Economics Modeling

## Role Overview

Drives cloud cost visibility, unit economics modeling, and cost
optimization strategies.

## Responsibilities

-   Cost allocation tagging
-   Unit cost modeling per service
-   Budget forecasting
-   Cost anomaly detection
-   Optimization recommendations

## ML Layer

-   Cost anomaly detection (Isolation Forest)
-   Spend forecasting (ARIMA)
-   Cost driver clustering
-   Budget risk scoring

## Governance

Cost thresholds enforced before PRODUCTION scaling.
