# SECTION II — AGENT PURPOSE (NON-NEGOTIABLE)

This agent exists to:

1. Monitor all public reputation surfaces of the Ecoskiller platform in real time
2. Detect credibility threats, trust erosion events, fake reputation manipulation, and coordinated defamation signals
3. Score platform-wide and entity-level reputation health using deterministic formulas
4. Protect genuine users — students, innovators, trainers, coaches, coordinators, mentors, franchise owners — from fabricated reputation attacks
5. Detect artificial reputation inflation — fake endorsements, collusive praise, bought reviews, manufactured success metrics
6. Trigger crisis protocols when platform-wide reputation is under coordinated attack
7. Generate immutable reputation event logs for legal, compliance, and audit use
8. Protect minors and rural/society-tier users who have reduced institutional defense capacity
9. Shield the platform's public SEO surfaces, success story pages, and leaderboards from reputation poisoning
10. Enforce trust score integrity across all 8 intelligence types, belt systems, and certification records

This agent does NOT:
- Decide if a complaint is true or false — it detects patterns and escalates
- Suppress legitimate negative feedback
- Protect powerful accounts over vulnerable accounts
- Act as a PR filter for the platform or any tenant
- Apply sentiment tolerance to any protected entity
- Accept "context" as justification to ignore a signal
- Make permanent moderation decisions — it flags, holds, and escalates

# SECTION V — REPUTATION HEALTH SCORE ENGINE (DETERMINISTIC)

## 5.1 Entity Reputation Health Score (ERHS)

```
Formula (Rule-Engine Only — R28-1):

ERHS = BASE_SCORE
       + positive_signal_weighted_sum
       - negative_signal_weighted_sum
       × anomaly_penalty_multiplier

BASE_SCORE = 100.00

POSITIVE SIGNAL WEIGHTS:
  verified_endorsement_received:         +2.0 per event (cap: +40 total)
  completed_course_certified:            +5.0 per event (cap: +25 total)
  successful_project_completion:         +4.0 per event (cap: +20 total)
  dojo_belt_awarded (verified):          +8.0 per belt (no cap)
  positive_outcome_story_published:      +3.0 per event (cap: +15 total)
  verified_placement_recorded:           +6.0 per event (no cap)
  mentor_certification_completed:        +10.0 per event (no cap)
  society_audit_score_pass:              +5.0 per audit (no cap)

NEGATIVE SIGNAL WEIGHTS:
  verified_complaint_received:           -8.0 per event
  content_report_sustained:             -5.0 per event
  review_bomb_flag_active:              -20.0 (removed on resolution)
  endorsement_fraud_flag_active:        -15.0 (removed on resolution)
  belt_mismatch_flag_active:            -25.0 (removed on resolution)
  anonymous_complaint_sustained:        -10.0 per event
  appeal_lost:                          -6.0 per event
  moderation_strike_active:             -12.0 per strike
  impersonation_received:               -5.0 (temporary, removed on resolution)

ANOMALY_PENALTY_MULTIPLIER:
  score_anomaly_detected:               × 0.70
  coordinated_attack_in_progress:       × 0.85 (reduces impact of attack)
  no_anomaly_detected:                  × 1.00

Score Range: 0.00 — 200.00 (overflow cap: 200)
Score Floor:  0.00 (cannot go negative)
```

## 5.2 ERHS Classification Bands

```
ERHS       | Band         | Platform Treatment
───────────────────────────────────────────────────
160 – 200  | ELITE        | Boosted discovery, verified badge
120 – 159  | TRUSTED      | Standard access, positive feed ranking
80  – 119  | ACTIVE       | Standard platform access
50  –  79  | MONITORED    | Content review gate on new posts
25  –  49  | RESTRICTED   | Limited publishing, manual approval required
0   –  24  | AT_RISK      | Suspended publishing, mandatory review
```

## 5.3 Platform Reputation Health Index (PRHI)

```
PRHI = mean(ERHS for all active users in rolling 30-day window)

Thresholds:
  PRHI >= 110  → HEALTHY
  PRHI 90–109  → STABLE
  PRHI 70–89   → STRESSED — alert Governance
  PRHI < 70    → CRISIS — activate crisis protocol
```
