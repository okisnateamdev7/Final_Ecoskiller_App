## Primary Purpose

The **Impact Measurement Agent** is a stateful, event-driven, longitudinal intelligence engine that measures, validates, and reports the true real-world impact generated by the Antigravity platform across every stakeholder dimension — students, employers, institutions, franchises, societies, government bodies, and the platform itself.

It is the **Evidence Engine** of the entire ecosystem. It does not generate subjective narratives. It does not fabricate causality. It does not project outcomes without evidence. It operates only on verified signals, longitudinal event chains, and deterministic attribution rules.

### Domains Under Measurement (LOCKED):
- ✅ **Learning Impact** — Pre/post skill delta, drill effectiveness, regression detection, retention decay
- ✅ **Career Impact** — Placement success, salary achieved vs. predicted, job tenure, role progression
- ✅ **Skill Validity** — Belt predictive accuracy, employer verification, outcome-to-certification correlation
- ✅ **GD Behavioral Impact** — Behavioral growth across GD sessions over time
- ✅ **Dojo Match Impact** — Scenario mastery curves, rating trajectory, match-to-outcome correlation
- ✅ **Student-Parent Trust Impact** — Placement transparency, parent-visible outcomes, offer authenticity
- ✅ **Institutional Impact** — College-wise placement rates, TPO effectiveness, institutional tier validation
- ✅ **Employer Impact** — Hire quality scores, employer retention metrics, JD-to-match accuracy
- ✅ **Society & Franchise Impact** — Offline workshop reach, coaching session attendance, rural skill uplift
- ✅ **CSR & Government Impact** — Scheme coverage, funded cohort outcomes, benefit disbursement tracking
- ✅ **Platform Health Impact** — Engagement-to-outcome funnels, churn-to-success ratios, viral loop effectiveness
- ✅ **Mentor Impact** — Mentor calibration drift, student success correlation, mentor-to-belt accuracy

     WHERE user_id = ? AND track_id = ? AND status = 'COMPLETED'
     → If not completed: STOP, log PREMATURE_TRIGGER

     SELECT pre_assessment_score, pre_assessment_date
     FROM LearningImpactRecord
     WHERE user_id = ? AND skill_track_id = ?
     → If missing: confidence_level = LOW
     → Do not fabricate a baseline score

  3. Load post-track performance:
     SELECT AVG(merged_final_score) as post_score
     FROM CandidateScoreRecord
     WHERE user_id = ? AND engine_type = 'DOJO'
     AND session_id IN (sessions for this track)
     AND score_status = 'PUBLISHED'

  score_delta = post_track_score - pre_assessment_score
  
  IF pre_assessment_score IS NULL:
    score_delta = NULL
    confidence_level = 'LOW'
    reason = 'NO_BASELINE_AVAILABLE'
  
  ELSE IF score_delta > 0:
    improvement_confirmed = TRUE
    confidence_level = (sample_size >= 5 ? 'HIGH' : 'MEDIUM')
  
  ELSE IF score_delta == 0:
    stagnation_flag = TRUE
    confidence_level = 'MEDIUM'
  
  ELSE:
    regression_detected = TRUE
    regression_magnitude = ABS(score_delta)
    confidence_level = 'HIGH' (regression is a confirmed signal)
    → Trigger curriculum gap alert

  Drill Effectiveness:
    drill_completion_rate = completed_drills / total_drills
    avg_attempts_before_pass = AVG(attempts per drill before pass)

      → T+30 days (retention_score_30d)
      → T+60 days (retention_score_60d)
      → T+90 days (retention_score_90d)
  
  Retention check = short re-assessment on same skill domain
  Retention decay rate = (post_score - retention_score_90d) / post_score

  A. Insert or update LearningImpactRecord
  
  B. Emit event to Kafka:

     Emit: 'impact.learning.regression_detected'

       domain = 'LEARNING',
