BASE_XP_FORMULA:
═════════════════════════════════════════════════════════════════

Base_XP = Category-specific base value [25–200]
  ├─ Assessment: 100
  ├─ Advancement: 200
  ├─ Streak: 50
  ├─ Mentorship: 25
  └─ Benchmark: 150

Difficulty_Multiplier = 1 + (0.10 × Difficulty_Score)
  ├─ Easy scenario (pass_rate >75%): 1.00 (×1.0)
  ├─ Normal scenario (pass_rate 40–60%): 1.05 (×1.05)
  ├─ Hard scenario (pass_rate <40%): 1.10 (×1.1)
  └─ Master scenario (pass_rate <15%): 1.15 (×1.15)
  ├─ Capped at 1.20 (no extreme multipliers)
  └─ Purpose: Reward users for tackling harder content

Performance_Multiplier = 0.5 + (0.005 × Performance_Score)
  ├─ Performance_Score = Participant score (0–100)
  ├─ Formula produces: 0.5 to 1.0 range
  ├─ Score 0%: 0.5× (half XP for low performance)
  ├─ Score 50%: 0.75× (three-quarter XP for median)
  ├─ Score 100%: 1.0× (full XP for perfect performance)
  └─ Purpose: Reward excellence, encourage high performance

Confidence_Multiplier = Confidence_Interval_Tightness
  ├─ High confidence (interval <8 points): 1.0×
  ├─ Moderate confidence (interval 8–15 points): 0.95×
  ├─ Low confidence (interval 15–20 points): 0.90×
  └─ Insufficient confidence (interval >20 points): 0.85×
  └─ Purpose: Penalize uncertain assessments (unreliable)

Anti_Inflation_Factor = Velocity_Adjustment × Saturation_Adjustment
  
  Velocity_Adjustment = 1 - (0.02 × Days_Played_This_Month)
  ├─ Decreases by 2% for each day active in month
  ├─ Day 1: 1.00× (baseline)
  ├─ Day 15: 0.70× (70% of XP)
  ├─ Day 30: 0.40× (40% of XP)
  └─ Purpose: Prevent grinding, encourage pacing

  Saturation_Adjustment = 1 - (0.001 × (Total_XP - Saturation_Point))
  ├─ Triggers when Total_XP exceeds Saturation_Point
  ├─ Saturation_Point = 2000 XP per skill
  ├─ Beyond saturation: 0.1% decrease per excess XP point
  ├─ At 3000 XP: 1 - (0.001 × 1000) = 0.90×
  ├─ At 5000 XP: 1 - (0.001 × 3000) = 0.70×
  └─ Purpose: Prevent infinite progression, encourage new skills

CAP RULES (LOCKED):

├─ Daily XP cap: 2000 XP per day (hard limit)
├─ Weekly XP cap: 10000 XP per week (hard limit)

└─ Purpose: Prevent exploitation, maintain economy health
