```

When multiple jurisdictions' laws apply:

1. MOST RESTRICTIVE RULE APPLIES
   ├─ If GDPR (EU) AND CCPA (CA) both apply → GDPR enforced
   ├─ Reason: Provides strongest user protection
   └─ Agent enforces higher standard across all users

2. ITAR/EAR RESTRICTIONS (OVERRIDES OTHERS)
   ├─ If US export control prohibits data transfer → Prohibited globally
   ├─ No user location or consent can override
   └─ Agent blocks all non-compliant operations

3. USER'S JURISDICTION LAW
   ├─ Student in EU: GDPR applies to personal data
   ├─ Recruiter in India: India privacy laws apply
   └─ Agent applies jurisdiction-specific procedures

4. PLATFORM'S JURISDICTION
   ├─ Platform registered in US: US law applies to operations
   ├─ State law: Delaware (or incorporated state)
   └─ Agent applies home jurisdiction to disputes

PRACTICAL EXAMPLE:
EU student (GDPR) + US platform (CCPA) + Anti-gravity data (ITAR)
→ Apply GDPR + CCPA (stricter than each alone) + ITAR restrictions
→ Result: Highest level of protection on all fronts
```
