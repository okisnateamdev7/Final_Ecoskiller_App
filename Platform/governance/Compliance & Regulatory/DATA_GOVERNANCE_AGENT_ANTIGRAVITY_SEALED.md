#### **L5: PUBLIC** (No Confidentiality)
- Public knowledge, marketing, transparency, metrics
- Access: Public (no auth), Read-only, Standard web security

- Retention: As needed for availability, Indefinite if evergreen

#### **L6: TEMPORARY** (Short-Lived)
- Session data, cache, computation results, buffers
- Retention: Session duration only, Auto-deletion, No backup

#### **L7: AUDIT ONLY** (Regulatory Records)
- Access logs, governance decisions, incident reports, compliance evidence
- Retention: 10+ years, Immutable, No modification, Cryptographic verification

## X. DATA BREACH DETECTION & RESPONSE

### 10.1 Breach Detection Mechanisms

**Behavioral Anomaly Detection:**
- Access at unusual hours (3 AM for day worker)
- Bulk downloads unusual for role
- Access from new/suspicious location
- Failed attempts exceeding threshold
- Privilege escalation without approval
- Cross-tenant access attempts
- Rapid sequential queries
- Suspicious API calls

**Response:** Session termination + MFA challenge + CRITICAL ticket + Account lock + Audit trail marked

### 10.2 Data Breach Response Procedure

**Phase 1: Detection & Immediate Response (<5 min)**
- Breach alert triggered
- Incident commander assigned
- Affected systems isolated
- Attacker access revoked
- Forensic evidence preserved
- Executive notification initiated

**Phase 2: Investigation (1-24 hours)**
- Determine data scope
- Calculate exfiltrated volume
- Assess dwell time
- Identify attack vector
- Check lateral movement
- Assess impact per user/tenant
- Evaluate attack sophistication

**Phase 3: Containment & Recovery (1-7 days)**
- Patch vulnerability
- Revoke credentials
- Rotate encryption keys
- Block attacker IP/domains
- Update WAF/IDS
- Implement monitoring
- Restore from backup
- Verify integrity
- Resume operations

**Phase 4: Notification & Compliance (24-72 hours)**
- Notify affected users
- Notify regulatory bodies
- Notify business partners
- Notify insurance carrier
- Public disclosure (if material)
- Board notification

**Phase 5: Post-Incident (1-3 months)**
- Complete forensic analysis
- Comprehensive incident report
- Post-mortem with stakeholders
- Root cause remediation
- Preventive control implementation
- Security training updates
- Policy enhancements
