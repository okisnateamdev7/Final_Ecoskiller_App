## 2️⃣ PURPOSE DECLARATION

### 2.1 Problem Solved

The USER_RIGHTS_EXPLANATION_AGENT resolves the following critical trust and accountability gaps in Ecoskiller Antigravity:

- **Rights Opacity**: Users unable to understand what data the platform holds, why, and for how long
- **Consent Confusion**: Recording consent, TOS acceptance, and data-use consent presented in legal language users cannot act on
- **AI Black Box**: Match scores, skill gap recommendations, and job placement probabilities produced without explanation — eroding trust
- **Grievance Paralysis**: Users unable to locate the correct escalation path for score disputes, certification rejections, offer letter disputes, or anti-cheat violations
- **Parent Visibility Gap**: Parents unable to understand what they can see, what they cannot, and why
- **Minor Protection Unawareness**: Students under age threshold unaware of special protections applicable to their data
- **Jurisdiction Blindness**: Users in India vs EU vs other jurisdictions receiving identical generic explanations that do not reflect their actual legal rights
- **Deletion/Export Confusion**: Users unable to understand what happens when they request deletion, what is preserved legally, and what timelines apply
- **Appeal Path Ambiguity**: Users who receive adverse AI or evaluator decisions not told how to appeal, what evidence they can access, and what timelines apply
- **Notification Rights Gap**: Users unaware they can configure, pause, or withdraw notification consent categories independently

## 9️⃣ FAILURE POLICY

### Failure Scenarios & Responses

| Failure Type | Severity | Response |
|---|---|---|
| Invalid input (schema violation) | P2 | STOP → LOG_VALIDATION_FAILURE → REJECT → RETURN_USER_SAFE_ERROR |
| JWT invalid or expired | P1 | STOP → LOG_AUTH_FAILURE → REJECT → PROMPT_REAUTH |
| Unknown request_type | P2 | STOP → LOG → REJECT → RETURN_TYPE_NOT_SUPPORTED response |
| Jurisdiction not in registry | P1 | STOP → LOG → ESCALATE_TO: GOVERNANCE_AGENT → RETURN_JURISDICTION_UNKNOWN |
| Minor safety ML gate failed | P1 | STOP_AI_OUTPUT → LOG → SERVE_TEMPLATE_FALLBACK → ALERT_GOVERNANCE |
| Reading level > grade 8 | P2 | REJECT_AI_OUTPUT → REGENERATE (max 2 retries) → TEMPLATE_FALLBACK on exhaustion |
| AI timeout (> 12s) | P2 | SERVE_TEMPLATE_FALLBACK → SET narrative_ai_unavailable=true → LOG |
| Grievance submission failure | P1 | STOP → LOG → RETAIN_USER_INPUT → RETRY (3x) → DEAD_LETTER + user acknowledgement |
| Erasure/Export request failure | P0 | LOG_IMMEDIATELY → ESCALATE_TO: COMPLIANCE_AGENT + GOVERNANCE_AGENT → HOLD_SLA_CLOCK |
| Parent-child link unverified | P1 | STOP → LOG → REJECT → PROMPT_IDENTITY_REVERIFICATION |
| Cross-tenant query attempt | P0 | IMMEDIATE_HALT → LOG_SECURITY_INCIDENT → ALERT_SECURITY_AGENT |
| Tone check failed (dismissive/evasive output) | P2 | REJECT_OUTPUT → REGENERATE → TEMPLATE_FALLBACK after 2 failures → LOG |
| COMPLIANCE_AGENT unreachable for erasure | P0 | HOLD_REQUEST → LOG → ALERT_OPS → HOLD_SLA_CLOCK → NOTIFY_USER_OF_DELAY |
